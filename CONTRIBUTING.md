# Contributing

Thank you for contributing to saas-backend. Please follow the checklist below before opening a pull request.

## PR Checklist

- [ ] Branch from `main` and keep your branch up to date with it
- [ ] Ensure all tests pass locally (`go test ./...` or the subset in `CLAUDE.md`)
- [ ] Add a CHANGELOG entry describing your change
- [ ] Request a review from at least one maintainer
